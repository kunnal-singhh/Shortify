# Shortify - URL Shortener Service

## Problem Statement
Long URLs are hard to share. There is a need for a simple, self-hosted URL shortener service with basic analytics like click tracking.

## Approach Explanation
Shortify is built to be a fast and reliable RESTful API. 
- **Tech Stack**: Built using modern backend technologies (Node.js/Express + Database).
- **Architecture**: A simple MVC pattern is used. It handles URL shortening and redirection efficiently, whilst tracking clicks in real-time.
- **Error Handling**: Implements strict validation and proper 404 responses for non-existent codes.

## Setup Instructions
1. Clone the repository.
2. Install dependencies by running `npm install` inside the `backend` directory.
3. Configure your environment variables in a `.env` file (e.g., `PORT`, `MONGO_URI`).
4. Start the development server using `npm start` or `npm run dev`.
5. The API will be available at `http://localhost:8000` (or your configured port).

## API Documentation

This is the API documentation for the Shortify URL Shortener service.

## Base URL
`http://localhost:<PORT>` (Default: `http://localhost:8000`)

---

## 1. Shorten a URL
Accepts a long URL and returns a unique short code.

**Endpoint:** `/shorten`
**Method:** `POST`
**Content-Type:** `application/json`

### Request Body
```json
{
  "url": "https://www.example.com/some/very/long/url"
}
```

### Success Response (200 OK)
```json
{
  "id": "xyz123"
}
```

### Error Response (400 Bad Request)
```json
{
  "error": "url is required"
}
```

---

## 2. Redirect to Original URL
Redirects the user to the original long URL associated with the short code.

**Endpoint:** `/:code`
**Method:** `GET`

### Path Parameters
- `code`: The short code generated by the `/shorten` endpoint (e.g., `xyz123`).

### Success Response
- **302 Redirect** to the original URL.

### Error Response (404 Not Found)
```json
{
  "error": "Short URL not found"
}
```

---

## 3. Get URL Analytics
Returns the click count and timestamp of each visit for a specific short URL.

**Endpoint:** `/analytics/:shortID`
**Method:** `GET`

### Path Parameters
- `shortID`: The short code of the URL.

### Success Response (200 OK)
```json
{
  "totalClicks": 5,
  "analytics": [
    { "timestamp": 1698765432100, "_id": "..." },
    { "timestamp": 1698765445000, "_id": "..." }
  ]
}
```

### Error Response (404 Not Found)
```json
{
  "error": "Short URL not found"
}
```

---

## 4. Get All URLs
Returns a list of all shortened URLs and their analytics summary.

**Endpoint:** `/urls`
**Method:** `GET`

### Success Response (200 OK)
```json
{
  "urls": [
    {
      "id": "65b...123",
      "shortId": "xyz123",
      "redirectURL": "https://www.example.com...",
      "shortURL": "http://localhost:8000/xyz123",
      "totalClicks": 5,
      "createdAt": "2024-01-01T12:00:00.000Z"
    }
  ]
}
```
